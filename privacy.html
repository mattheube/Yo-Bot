Politique de confidentialité

Dernière mise à jour : 30 juin 2026

Cette politique de confidentialité explique quelles données sont collectées, utilisées et stockées lorsque vous utilisez Yokai Watch Bot sur Discord.

1. Données collectées

Yokai Watch Bot peut collecter et stocker certaines données nécessaires au bon fonctionnement du bot, notamment :

  • l'identifiant Discord de l'utilisateur ;
  • l'identifiant du serveur Discord ;
  • la langue choisie par l'utilisateur ;
  • l'inventaire du joueur ;
  • les Yo-kai possédés ;
  • les objets possédés ;
  • les échanges entre joueurs ;
  • certaines informations liées à la progression dans le jeu ;
  • certaines commandes utilisées avec le bot, si cela est nécessaire pour le fonctionnement ou la sécurité du service.

2. Utilisation des données

Les données collectées sont utilisées uniquement pour faire fonctionner les fonctionnalités du bot, notamment :

  • sauvegarder la progression des joueurs ;
  • gérer les profils, inventaires, Yo-kai et objets ;
  • permettre les échanges entre utilisateurs ;
  • gérer les préférences de langue ;
  • assurer la sécurité et la stabilité du bot ;
  • corriger des bugs ou empêcher les abus.

3. Données non collectées

Yokai Watch Bot ne collecte pas :

  • les mots de passe ;
  • les adresses e-mail privées ;
  • les informations bancaires ;
  • les messages privés des utilisateurs ;
  • les informations personnelles sensibles qui ne sont pas nécessaires au fonctionnement du bot.

4. Messages Discord

Yokai Watch Bot fonctionne principalement avec des commandes Discord. Le bot ne lit pas les messages privés des utilisateurs. Si certaines interactions publiques sont utilisées, elles le sont uniquement pour répondre aux commandes ou assurer le fonctionnement normal du bot.

5. Partage des données

Les données collectées par Yokai Watch Bot ne sont pas vendues. Elles ne sont pas partagées avec des tiers, sauf si cela est nécessaire pour respecter une obligation légale, protéger le service ou assurer la sécurité des utilisateurs.

6. Conservation des données

Les données sont conservées aussi longtemps que nécessaire pour permettre le fonctionnement du bot. Certaines données peuvent être supprimées si elles ne sont plus utiles, si le bot est retiré d'un serveur, ou si un utilisateur demande leur suppression.

7. Suppression des données

Un utilisateur peut demander la suppression de ses données en contactant le propriétaire du bot sur Discord. La demande doit permettre d'identifier clairement le compte Discord concerné.

8. Sécurité

Des mesures raisonnables sont prises pour protéger les données stockées par Yokai Watch Bot. Cependant, aucun système informatique n'est totalement sécurisé, et aucune garantie absolue ne peut être donnée.

9. Modifications de cette politique

Cette politique de confidentialité peut être modifiée à tout moment. En cas de modification importante, la date de mise à jour sera changée.

10. Contact

Pour toute question concernant cette politique de confidentialité ou pour demander la suppression de vos données, veuillez contacter le propriétaire du bot sur Discord.
